# MD5

Pure Julia implementation of MD5 hash algorithm. Inspired by 
[SHA.jl](https://github.com/staticfloat/SHA.jl/blob/master/src/common.jl).

AT THIS POINT USES A LOT OF COPY PASTED CODE FROM SHA.jl
